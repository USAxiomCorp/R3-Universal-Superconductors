# CONTRIBUTING TO R3 UNIVERSAL SUPERCONDUCTORS

## Ways to Contribute

### 1. Verification Attempts
The most valuable contribution. Attempt to synthesize and measure any material in this repository. Report all results — positive and negative.

### 2. Protocol Improvements
If you find a better way to grow, dope, or measure these materials, submit an improvement.

### 3. Error Reports
Found an inconsistency? A missing step? An incorrect parameter? Tell us.

### 4. New Materials
If you have a superconductor specification refined by R³, consider contributing it.

---

## How to Submit

1. Fork the repository
2. Create a branch for your contribution
3. Make your changes
4. Submit a pull request

For verification results, you can also email directly: verification@advanceerivs.com

---

## What Happens Next

All verified results become part of the permanent R³ Truth Chain record. Contributors are credited in the repository and in the immutable record.

---

## Code of Conduct

Be respectful. Be scientific. Be honest. Negative results are as valuable as positive ones.
